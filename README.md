# Java Array Operations

Java Array Operations adalah proyek yang mengeksplorasi manipulasi data dalam struktur data Array satu dimensi. Program ini mencakup berbagai operasi dasar yang penting untuk memahami bagaimana data disimpan dan dikelola dalam memori menggunakan Java.

## 🌟 Fitur Utama

- **Input & Output Array:** Mengisi elemen array melalui input pengguna dan menampilkannya kembali.
- **Pencarian (Searching):** Mencari elemen tertentu di dalam deret array.
- **Pengolahan Data:** Menghitung total nilai, rata-rata, atau menemukan nilai tertinggi/terendah dalam array.
- **Iterasi Efisien:** Menggunakan `for` dan `for-each` loops untuk memproses data secara sistematis.



## 🧠 Konsep yang Dipelajari

1. **Deklarasi & Inisialisasi:** Cara memesan ruang memori untuk tipe data tertentu.
2. **Indexing:** Memahami bahwa index array dimulai dari `0`.
3. **Array Length:** Menggunakan property `.length` untuk iterasi dinamis.

## 🛠️ Teknologi

- **Bahasa:** Java
- **JDK:** Version 8 atau yang lebih baru.
